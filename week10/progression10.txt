document.addEventListener("DOMContentLoaded", () =>)
event นี้จะทำการก็ต่อเมื่อเว็บโหลดเสร็จและ js พร้อมทำงาน

ต่อมาก็ใช้ dom ในการเข้าถึงตัวแปรแต่ละตัว และผมก็จะประกาสตัวแปรเอาไว้เก็บค่า default ไว้ก่อน
และประกาสตัวแปรเพ่ิมเป็นตัวที่เช็คว่ามีข้อมูลใน localStorage ไหมถ้ามีก็เอาตัวนั้นแต่ถ้าไม่มีก็เป็นค่า default แทนทำให้ช่วยในการ
เช็คและเข้าถึงตัวแปรได้สะดวกและง่าย และลดการเขียนโค้ดซ้ำๆ ซึ่งทุกครั้งที่หน้าเว็บโหลดก็จะแสดงค่าที่อยู่ใน local แต่ถ้าใน local 
ไม่มีก็จะเอาค่า default มาแทน

โดยในปุ่ม save จะ addEventListener เข้าไปเป็น type click พอมีการคลิกจะทำการเอาค่าใน input ไปเก็บบน localStorage
เสร็จแล้วก็แสดงบนหน้าเว็บ 
ต่อมาปุ่ม reset ก็จะมี addEventListener เหมือนกัน แต่แทนที่จะ set ข้อมูลใน local เปลี่ยนเป็น remove แทนเพื่อให้เข้าเงื่อนไข
เวลาโหลดเว็บใหม่(ถ้าไม่มีข้อมูบลใน localStorage ก็จะเอาค่า default มาแทน) เสร็จแล้วก็แสดง background , color , fontsize
เป็นค่า default ที่ประกาสไว้และ เข้าถึง value ใน input ให้มีค่าเป็น default เหมือนเป็นการ clear นั้นเอง
